# Что это?

Данный модуль добавляет на сервер команду `/blind` и дублирующие её `/bs`, `/black`, `/blackscreen`.
При вызове команды экран игрока становится чёрным на определенноё время.
Это необходимо ролевым игрокам для вырезки текста из чата и создания скриншотситуаций.

Чтобы скрипт работал - достаточно просто его подключить после основных включений.
```pawn
#include <blind>
```

# Совместимость и настройка
Для работы необходим `Pawn.CMD`, но вы с легкостью можете адаптировать включение и под другие коммандные процессоры. Вам лишь необходимо изменить объявления команды `CMD:blind`.

Время, на которое появляется чёрный экран регулируется с помощью определения:
```pawn
#define blind_Time(%0) 3000
```
`%0` - определяет ID игрока, с помощью которого в случае надобности вы можете извлечь индивидуальное время срабатывания, установленное игроком.
